# 标题日记

一个纯文本 web 日记应用，专注记录生活琐事。  
[http://kylebing.cn/diary](http://kylebing.cn/diary)



## 一、界面

![main_view](https://github.com/user-attachments/assets/4c3efa8f-322c-41ae-bb0d-0b20c6143bea)


## 二、用法

**标题**：20 字以内，概括  
**内容**：事件详情

记录一天中你感觉值得记录的事，生活、学习、工作、运动 等等方面。  
记下此刻，记下你怕忘的内容，日后好查找。
